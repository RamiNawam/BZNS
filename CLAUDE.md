# CLAUDE.md — Micro-Business Launchpad (BZNS)

## Project Overview

An AI copilot that takes someone from "I have a skill and want to start a business" to "I am legally operating a business in Québec." All the steps, all the money, all the numbers, in one place, in plain language, in any language.

**Hackathon:** Claude Builders Hackathon @ McGill — April 1-4, 2026
**Track:** Economic Empowerment & Education (Social Impact)
**Team:** 3 people. This developer (Person A) owns Frontend + Infrastructure.

## The Problem

Québec has one of the most fragmented startup environments in Canada. To legally start a micro-business, you need to navigate 6+ government entities, 80+ funding programs, and a tax system that surprises every first-time self-employed person. The information exists but is scattered across dozens of websites, mostly in bureaucratic French.

## Target Users

- **Yara, 26** — Syrian immigrant, pastry chef. Wants to sell baked goods from her home kitchen. Eligible for $90K+ in funding she doesn't know about.
- **Marcus, 23** — McGill grad, freelance software consultant. Doesn't know about REQ registration or 12.8% QPP double contributions.
- **Fatima, 41** — Single mother wants to open a home daycare. Doesn't know about the STA program that provides income support during startup.

## Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** TailwindCSS
- **State:** Zustand (3 stores: useProfileStore, useRoadmapStore, useChatStore)
- **Database + Auth:** Supabase (Postgres, magic-link auth, RLS, file storage)
- **AI:** Claude API — Sonnet 4.6 (profile synthesis, roadmap generation, financial insights, contextual assistant)
- **Validation:** Zod (every Claude response validated before storage)
- **Translations:** i18next (static UI strings FR/EN; AI responses multilingual natively)
- **Documents:** docx npm package (stretch feature)
- **Deploy:** Vercel

## Core Features

1. **Conversational Intake** — 8 plain-language questions, one per screen, progress bar. Builds structured user profile. Claude classifies business type from free text.
2. **Legal Roadmap** — Personalized, dependency-ordered checklist of every legal step. Generated by Claude from curated knowledge base. Each step has cost, timeline, documents, government links.
3. **Funding Matcher** — Deterministic scoring engine (no AI) ranking 15+ Quebec funding programs. Claude only for "explain this program." Key demo moment: "$95,000+ available."
4. **Financial Snapshot** — Hybrid: deterministic tax math (GST/QST, QPP 12.8%, income tax brackets) + Claude for situation-specific insights. Shows gross revenue → taxes → take-home. Key demo moment: "$573/mo take-home."
5. **Contextual AI Assistant** — Persistent chat that knows full user state. Cites sources. Refers to professionals when out of scope.

### Stretch Features
6. **Starter Kit** — One-click .docx generation (contract, invoice, pitch). Legal clauses hardcoded, Claude fills personalized fields.
7. **Pricing Engine** — Pure math, no AI. Lowest priority.

## Key Constraints

- **No separate backend** — Next.js API routes handle everything server-side
- **No ORM** — Supabase JS client for all database operations
- **Every Claude response must be validated with Zod** before reaching the user
- **Tax math is deterministic TypeScript** — NOT AI-generated
- **Knowledge base is flat JSON files in /data/** — loaded at build time
- **Demo account (Yara) must be pre-loaded** — judges never see a spinner
- **All AI content is multilingual natively** — Claude responds in whatever language the user writes
- **Static UI strings translated via i18next** — FR/EN only
